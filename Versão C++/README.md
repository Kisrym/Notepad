# Versão C++


## Requisitos

Para rodar o projeto é necessário:

* CMake 3.5+
* QtCreator (por enquanto)

## Como rodar

Tendo o QtCreator, basta abrir o projeto (selecionando todos os arquivos, principalmente o *CMakeLists.txt*)
